# Hands-on Tutorials on JWST Data Analysis for the [International Virtual Course on Astronomy Institut Teknologi Bandung 2022](https://www.as.itb.ac.id/ivcas2022/)  

This repository contains materials for a tutorial on the JWST NIRCam imaging data analysis. The tutorial will be given as part of the [IVC Astronomy ITB 2022](https://www.as.itb.ac.id/ivcas2022/) on August 16, 2022.
